## 2. 공통 컴포넌트·상태

### 헤더 & 네비게이션
- **로고**: MediaPulse (primary-600)
- **네비게이션**: Today Hot, Explore, Timeline, Export, Gallery
- **사용자 메뉴**: Feedback, Settings, Logout
- **모바일 햄버거**: 3선 메뉴, 접근성 aria-label="메뉴"

### 로딩 & 스켈레톤
- **스켈레톤**: 차트 영역, 테이블 행, 카드 컴포넌트
- **로딩 상태**: aria-busy="true", 스피너 + "데이터 로딩 중..."
- **실시간 업데이트**: Socket 연결 상태 표시

### 에러 처리
- **4xx 에러**: "요청 정보를 확인해주세요" + 홈으로 버튼
- **5xx 에러**: "일시적 오류가 발생했습니다" + 새로고침 버튼
- **네트워크 에러**: "연결을 확인해주세요" + 재시도 버튼

### 알림 & 스낵바
- **성공**: 초록색, "완료되었습니다" 3초 자동 숨김
- **에러**: 빨간색, "오류가 발생했습니다" 5초 수동 닫기
- **정보**: 파란색, "업데이트되었습니다" 4초 자동 숨김
